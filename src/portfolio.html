Avec Portfolio, vous pourrez montrer les différents projets web sur que vous êtes en train de développer ou ceux sur lesquels vous avez travaillé. Ce site web statique et responsive a été développé avec TailwindCSS, un framework d'utilitaires CSS permettant de prototyper rapidement des layouts, et, selon votre préférence, vous pouvez utiliser comme module bundler Parcel ou Webpack.

Procédure de déploiement

Pour déployer le site, vous devez d'abord générer les fichiers via les modules bundler suivants:

  • Parcel : exécutez la commande
    npm run parcel
  • Webpack : exécutez la commande
    npm run webpack

Ensuite, il vous suffira de déployer le contenu du répertoire dist/ sur votre serveur Apache configuré comme il faut.
